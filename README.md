# Hortus Cafe Website

## Project Description
The Hortus Cafe website is a platform for showcasing the cafe’s menu, photos, events, and providing information to customers. It aims to enhance the customer experience with a user-friendly interface and engaging content.

## Features
- Menu display with prices and descriptions
- Event calendar
- User-friendly navigation
- Responsive design for mobile and desktop
- Contact form for inquiries
- Integration with social media

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Hosting:** Heroku

## Deployment Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/NovaWeb-Studio/hortus-cafe-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd hortus-cafe-website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
5. Open in your browser: `http://localhost:3000`
